# kissES
Keep It Sweet and Simple Elasticsearch ORM

## Maven

```xml
<repository>
    <id>git-chsfleury</id>
    <name>chsfleury's Git based repo</name>
    <url>https://raw.githubusercontent.com/chsfleury/kissES/master/maven</url>
</repository>
```

```xml
<dependencies>
    <dependency>
        <groupId>org.kisses</groupId>
        <artifactId>kisses-annotations</artifactId>
        <version>5.0.0.0-SNAPSHOT</version>
    </dependency>

    <dependency>
        <groupId>org.kisses</groupId>
        <artifactId>kisses-core</artifactId>
        <version>5.0.0.0-SNAPSHOT</version>
    </dependency>
</dependencies>
```
